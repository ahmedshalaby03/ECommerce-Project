# 🛒 ECommerce Project

A fully responsive e-commerce store built with pure HTML, CSS & Vanilla JavaScript.

## ✨ Features

- 🛍️ Dynamic product listing fetched from JSON
- 🛒 Cart system with localStorage
- ➕➖ Increase / Decrease / Remove items from cart
- 📦 Checkout page with order summary
- 📊 Orders sent directly to Google Sheets
- 📱 Fully responsive on all screen sizes

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Google Apps Script (Sheets integration)
